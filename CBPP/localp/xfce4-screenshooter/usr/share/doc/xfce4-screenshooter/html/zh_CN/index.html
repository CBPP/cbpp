Xfce4 屏幕截图工具简介

此应用程序让您截取整个屏幕、活动窗口或选中的区域。您可以设置截图前的延时和截图之后所要执行的的动作：保存截图为 PNG 文件、将截图复制至剪贴板、使用另一应用程序打开或上传至免费的在线图片提供服务 ZimageZ。

还有一个 Xfce 面板插件。

使用此应用程序

通过用户界面

要启动此应用程序，选择 Xfce 主菜单下的 “附件” 类别中的 “屏幕截图” 项目。

窗口会如下显示：

截取区域

“截取区域” 区域让您设置要截取的范围：

  • “全屏幕” 截取您看到的整个屏幕的截图。
  • “活动窗口” 截取活动窗口的截图。它是此对话框出现之前的活动窗口，或者，如果您设置了延时，则是延时时间之后的窗口。
  • “选择一个区域” 让您通过在松开鼠标按钮之前在您希望截取的屏幕区域点击并拉出一个矩形框选择要截取的区域。
截取指针

“截取鼠标指针” 选项让您选择截图是否包括鼠标指针。

截取前延时

“截取前延时” 区域让您设置按下 “确定” 按钮与截取截图开始之间经过的延时。此延时让您打开菜单或执行任何显示在截图上的您需要看到的动作。

截取后

按下 “确定” 按钮后会显示第二个窗口：

预览

“预览” 区域显示截图的缩略图。

动作

“动作” 区域让您选择要在截图上执行的操作。

保存

“保存” 选项将截图保存为 PNG 文件。会显示一保存对话框。您能够设置保存的位置和文件名。

Xfce4 屏幕截图工具同样能够在任何 GVfs 支持的远程文件系统上保存截图，比如 FTP、SAMBA、SFTP 或通过 SSH 连接的远程计算机...。您只需要使用 gvfs-connect 或 Gigolo 与远程计算机连接，它会出现在保存对话框的左侧栏上。

复制至剪贴板

“复制至剪贴板” 选项让您在其它应用程序中粘贴截图，比如文字处理器。此选项仅在剪贴板管理器运行时有效。

打开方式

“打开方式” 选项在系统的缓存目录中保存截图并用您从下拉列表中选择的应用程序打开它。在下拉列表之中会自动检测并添加支持图片的应用程序。

上传至 ZimageZ

“上传至 ZimageZ” 选项让您在免费的在线提供服务中上传您的截图，这样您就可以轻松地与其他人共享了。ZimageZ 自动给您上传的图片生成一个大幅和一个小幅缩略图，可以用来创建指向完整大小截图的缩略图。

按下 “确定” 按钮后，您会看到此对话框：

如果您未拥有一个 ZimageZ 用户名，您需要在 ZimageZ website 上创建一个用户。该网站支持多种语言，您可以使用右上角的菜单切换。您必须填写对话框中的所有项目。当您点击 “确定” 按钮后，上传就会开始。

下面的对话框会带给您完整大小截图的链接，大缩略图、小缩略图，还有创建指向完整大小截图的缩略图的 HTML 和 BBcode 模板：

通过命令行

命令行选项让您快速截取截图。它们同样让您配置大多数键盘右上角的 Print Screen 键以便截取截图。要这样做，配置您的桌面环境中的键盘绑定以便在按下 Prt Scrn 键时 xfce4-screenshooter 带以下一个或多个选项启动。

命令行选项

-w 选项

-w 选项让您截取活动窗口的截图。

-f 选项

-f 选项让您截取整个屏幕的截图。

-r 选项

-r 让您通过在松开鼠标按钮之前在您希望截取的屏幕区域点击并拉出一个矩形框选择要截取的区域。

-d 选项

在给定 -w、-f 或者-r 选项时 -d 选项后接一个正整数让您设置截取截图前的延时。

-s 选项

-s 选项后接一个已存在文件夹的路径让您设置保存截图的路径。此选项仅在给定 -w、-f 或者-r 选项时生效。

-o 选项

如果 -o 选项后接一个应用程序名称给定，会在系统的临时目录保存截图，并使用此应用程序打开截图。此选项仅在给定 -w、-f 或者 -r 选项时生效。

-u 选项

如果给定 -u 选项，截图会上传至 ZimageZ。查看上述获取详情。此选项仅在给定 -w、-f 或者 -r 选项时生效。

使用 Xfce 面板插件

添加插件

要在面板中添加插件，在面板上右击并选择 “添加新项目”。选择名称为 “截图” 的插件然后添加。

配置插件

您可以通过在插件上右击并选择 “属性” 配置截图如何截取。会出现一个与主应用程序的对话框相似的首选项对话框。请看看上述获取不同的首选项。

使用插件

要截取截图，仅在插件上点击。您可以在插件按钮上使用鼠标滚轮在要截取的不同区域间切换。
