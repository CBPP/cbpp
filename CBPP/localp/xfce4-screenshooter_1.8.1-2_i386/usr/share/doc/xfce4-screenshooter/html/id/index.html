Pengantar ke Xfce4 Screenshooter

Aplikasi ini memungkinkan anda untuk menangkap seluruh layar, jendela yang aktif atau daerah terpilih. Anda dapat mengatur jeda yang diperlukan sebelum cuplikan layar diambil dan aksi yang akan dilakukan setelahnya: menyimpannya dalam berkas PNG, salin ke papan klip, membukanya dengan aplikasi lain, atau menyimpannya di ZimageZ, sebuah layanan penyimpanan gambar daring.

Plugin untuk panel Xfce juga tersedia.

Menggunakan aplikasi

Via antarmuka pengguna

Untuk menjalankan aplikasi, gunakan item "Cuplikan Layar" di kategori "Aksesoris" dari menu utama Xfce.

Jendela ini akan tampak:

Daerah untuk ditangkap

Bagian "Daerah akan ditangkap" memungkinkan anda untuk mengatur cuplikan layar apa yang akan diambil:

  • "Seluruh layar" mengambil cuplikan layar dari seluruh layar yang anda lihat.
  • "Jendela aktif" mengambil cuplikan layar dari jendela yang aktif. Jendela yang aktif adalah jendela sebelum dialog ini muncul, atau jika anda mengatur jeda, jendela yang aktif setelah jeda.
  • "Pilih daerah" memungkinkan anda untuk memilih daerah yang akan ditangkap dengan klik pada titik di layar lalu menarik persegi ke area yang ingin anda tangkap, sebelum melepaskan tombol tetikus.
Menangkap penunjuk

Opsi "Tangkap penunjuk" memungkinkan anda untuk memilih apakah cuplikan layar akan menyertakan pula penunjuk tetikus.:

Jeda sebelum menangkap

Bagian "Jeda sebelum menangkap" memungkinkan anda untuk mengatur jeda yang akan berlangsung di antara menekan tombol "OK" dan mengambil cuplikan layar. Jeda ini akan memungkinkan anda untuk membuka menu atau melakukan aksi apapun yang ingin anda lihat nantinya pada cuplikan layar.

Setelah menangkap

Setelah menekan tombol "OK" jendela kedua akan ditampilkan::

Pratilik

Bagian "Pratilik" menampilkan miniatur cuplikan layar.

Aksi

Bagian "Aksi" memungkinkan anda untuk memilih apa yang harus dilakukan di cuplikan layar.

Simpan

Opsi "Simpan" akan menyimpan cuplikan layar pada berkas PNG. Sebuah dialog simpan akan ditampilkan. Anda dapat mengatur lokasi penyimpanan, dan memberi nama pada berkas.

Xfce4 Screenshooter juga dapat menyimpan cuplikan layar ke sistem berkas jarak jauh apapun yang didukung oleh GVfs, seperti FTP, SAMBA, SFTP, komputer jarak jauh yang dapat diakses via SSH... Anda hanya perlu menyambung ke sistem berkas jarak jauh ini menggunakan gvfs-connect atau Gigolo dan itu akan tersedia di kolom sebelah kiri dialog simpan.

Salin ke papan klip

Opsi "Salin ke papan klip" memungkinkan anda untuk merekatkan cuplikan layar ke aplikasi lain, seperti pengolah kata. Opsi ini hanya tersedia jika manajer papan klip berjalan.

Buka dengan

Opsi "Buka dengan" menyimpan cuplikan layar ke direktori sementara sistem dan membukanya dengan aplikasi yang dipilih dari senarai taruh bawah. Aplikasi yang mendukung citra akan secara otomatis terdeteksi dan ditambahkan ke senarai taruh bawah.

Simpan di ZimageZ

Opsi "Simpan di ZimageZ" memungkinkan anda untuk menyimpan cuplikan layar anda di layanan penyimpanan daring gratis, sehingga anda dapat membaginya dengan mudah kepada orang lain. ZimageZ secara otomatis membuat miniatur besar dan kecil dari cuplikan layar anda, yang dapat digunakan untuk membuat miniatur yang mengarah ke cuplikan layar ukuran penuh.

Setelah menekan tombol "OK", anda akan ditampilkan dialog ini:

Jika anda tidak memiliki nama pengguna ZimageZ, anda perlu membuatnya di situs web ZimageZ. Beberapa bahasa tersedia di situs web ini, anda dapat menggantinya menggunakan menu di kanan atas. Anda harus mengisi semua bagian di dialog ini. Ketika anda klik tombol "OK", pengunggahan akan dimulai.

Dialog di bawah akan memberikan anda tautan ke cuplikan layar ukuran penuh, miniatur besar, miniatur kecil, dan juga contoh HTML dan BBcode untuk membuat miniatur yang mengarah ke cuplikan layar ukuran penuh:

Via baris perintah

Opsi baris perintah memungkinkan anda untuk mengambil cuplikan layar secara cepat. Opsi baris perintah juga memungkinkan anda untuk mengatur tombol Print Screen yang berada di sebelah kanan atas pada kebanyakan papan ketik sehingga dapat mengambil cuplikan layar. Untuk melakukan ini, atur pengikat kunci lingkungan desktop anda sehingga akan menjalankan xfce4-screenshooter dengan satu atau beberapa opsi berikut ketika tombol Prt Scrn ditekan.

Opsi baris perintah

Opsi -w

Opsi -w memungkinkan anda untuk mengambil cuplikan layar dari jendela yang aktif.

Opsi -f

Opsi -f memungkinkan anda untuk mengambil cuplikan layar seluruh layar.

Opsi -r

Opsi -r memungkinkan anda untuk memilih daerah yang akan ditangkap dengan cara klik lalu menarik persegi ke area yang ingin anda tangkap, sebelum melepaskan tombol tetikus.

Opsi -d

Opsi -d yang diikuti dengan bilangan positif memungkinkan anda untuk mengatur jeda sebelum mengambil cuplikan layar ketika opsi -w, -f atau -r diberikan.

Opsi -s

Opsi -d yang diikuti dengan alamat ke folder yang telah ada memungkinkan anda untuk mengatur letak cuplikan layar akan disimpan. Opsi ini hanya mempunyai efek jika -w, -f atau -r diberikan.

Opsi -o

Jika opsi -o diberikan, diikuti dengan sebuah nama aplikasi, cuplikan layar akan disimpan ke direktori sementara sistem dan dibuka dengan aplikasi yang namanya akan diberikan setelah -o. Opsi ini hanya mempunyai efek jika opsi -w, -f atau -r diberikan.

Opsi -u

Jika opsi -u diberikan, cuplikan layar akan disimpan di ZimageZ. Lihat di atas untuk keterangan lebih lanjut. Opsi ini hanya mempunyai efek jika opsi -w, -f atau -r diberikan.

Menggunakan plugin Xfce panel

Menambah plugin

Untuk menambah plugin ke panel, klik kanan di panel lalu pilih "Tambah Item Baru". Pilih plugin bernama "Cuplikan Layar", lalu tambahkan.

Mengatur plugin

Anda dapat mengatur bagaimana cuplikan layar diambil dengan klik kanan pada plugin dan memilih "Properti". Dialog pengaturan yang mirip dengan dialog aplikasi utama akan muncul. Silakan lihat di atas untuk mengetahui perbedaan pengaturannya.

Menggunakan plugin

Untuk mengambil cuplikan layar, cukup klik pada plugin. Anda dapat menggunakan roda gulung di tombol plugin untuk berpindah di antara daerah yang berbeda untuk ditangkap.
